Cemex Valley - Web Application for CEMEX

School Project

This web application was intended to improve the performance of software developers at CEMEX based on KPIs by gamification through the pages and a videogame. Cemex Valley (the videogame) is integrated with the work of the software developers.
The gameplay is related to the tasks software developers need to do. Whenever a task was approved, they built a building. When there were many buildings, they ended up with a city. They also see how many cities they have built by wandering in the world.
Develop with Bootstrap 4, JavaScript, NodeJS, and SQL Server. Responsible for the videogame development in Unity and integration with the web.
